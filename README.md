# regsub-mcp
[![MCPize](https://mcpize.com/badge/@pubspro/regsub-mcp)](https://mcpize.com/mcp/regsub-mcp)
MCP server for regulatory submission intelligence — ICH guideline lookup, CTD module mapping, eCTD structure guidance, and FDA/EMA query support

## Connect via MCPize

Use this MCP server instantly with no local installation:

```bash
npx -y mcpize connect @pubspro/regsub-mcp --client claude
```

Or connect at: **https://mcpize.com/mcp/regsub-mcp**
